### Main Tecnologies

* React version 19.1
* Node version 24

### Running Library Frontend in development

* Clone repository;
* In root path of project build the containers:
```sh
docker compose build
```

* To run the application:
```sh
docker compose up
```

* Application runs in http://localhost:8080
